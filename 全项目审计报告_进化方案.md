## 一、项目概述

PredictEdge 是一个预测市场智能套利平台，通过聚合多个预测市场数据源，利用AI多因子分析模型，识别概率偏差和套利机会。

**技术栈：**
- 后端：FastAPI + Python 3.10+ + SQLite
- 前端：原生HTML/CSS/JS + Tailwind CSS (CDN)
- 数据源：Polymarket Gamma API + Manifold Markets + Kalshi API
- 认证：JWT + bcrypt
- 部署：单服务器部署

## 五、性能分析

### 5.1 当前性能

- 事件加载：首次~3s（三源并行获取）
- 缓存刷新：每5分钟一次
- 内存占用：~50MB
- 并发能力：SQLite支持~100并发读

### 5.2 性能瓶颈

1. **数据源API延迟**：外部API响应时间1-3s
   - 已优化：并行获取 + 内存缓存
   - 未来优化：增加Redis缓存层

2. **SQLite限制**：写入并发低
   - 当前阶段：用户量小，完全够用
   - 未来优化：迁移到PostgreSQL
